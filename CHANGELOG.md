## [Initial Release] - 2026-03-26

### Added
- Search Magic: The Gathering cards using Scryfall's full search syntax
- Card detail view with image, oracle text, mana cost, type, and flavor text
- Scryfall Tagger integration — browse oracle tags and art tags per card
- Search by tag in-app directly from the tags view
- Saved Cards command — bookmark cards and access them later
- Sort results by name, EDHRec rank, or price
- Multi-select cards to copy names or open a group search on Scryfall
- Copy card image to clipboard
- Links to Scryfall, EDHRec, and Scryfall Tagger from every card
